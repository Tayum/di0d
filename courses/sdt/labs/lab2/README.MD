Lab 2
============
Task:
---
1. Описати однотипні дані на прикладі обраної предметної області. Представити їх масивом.
2. Обрати (обґрунтувати вибір) та описати алгоритм сортування масиву.
3. Побудувати та описати логічну модель динамічної структури програми сортування однотипних даних за допомогою діаграми діяльності в нотації UML.
4. Реалізувати логічну модель динамічної структури програми сортування даних мовою Java.
5. Навести приклад використання програми сортування даних.
